kupeck's font library


Instead doing something normal, I here at kupeck's font library give you one line of css or html code, and you paste that into your website, and the fonts just work!


CSS: @import url('https://raw.githubusercontent.com/kupeck/fonts/master/fonts.css');

HTML: <link rel="stylesheet" href="https://raw.githubusercontent.com/kupeck/fonts/master/fonts.css">

CSS rules to specify families: font-family: '***'; replace *** with the font's name
